Aalto University JSAV/OpenDSA exercises 2023

These visual algorithm simulation exercises have been developed at Learning + Technology research group at Aalto University during years 2021-2023. The exercises are implemented with JSAV and developed further from corresponding exercises in the OpenDSA electronic textbook.

Depth-first search and Breadth-first search

  • Semi-rectangular grid (avoid confusing edge overlap)
  • Random graph
  • A disconnected component
  • Pseudocode inside exercise

Kruskal's algorithm

  • Semi-rectangular grid (avoid confusing edge overlap)
  • Random graph
  • Uniform color scheme with DFS & BFS

Scaffolded Prim's algorithm

  • Priority queue interface: enqueue, dequeue, update operations
  • Color legend
  • Model answer highlights comparison operations

Non-Scaffolded Prim's algorithm

  • Pseudocode with highlighted GUI operations
  • Student clicks an edge to add it to the spanning tree

Scaffolded Dijkstra's algorithm v1

  • Used at autumn 2022 experiment
  • Priority queue interface: enqueue, dequeue, update operations
  • Color legend to highlight algorithmic concepts
  • Semi-rectangular grid
  • Exercise instances generated from a template
    • A disconnected component
    • Two edge relaxation situations
    • Two situations where a relaxation is not required
    • The student's solution can be remapped to the template
  • Most recently dequeued vertex is highlighted

Scaffolded Dijkstra's algorithm v2

  • Will be used at autumn 2023
  • Similar to Scaffolded Dijkstra's algorithm v1
  • Both vertices and edges are colored according to algorithmic concepts

Non-Scaffolded Dijkstra's algorithm

  • Pseudocode with highlighted GUI operations
  • Student clicks an edge to add it to the spanning tree
  • Hard to do without pen and paper notes
